2. Explique BIOS com suas palavras

Sigla que significa "Basic Input/Output System", a BIOS é um chip minúsculo localizada normalmente nas bordas da placa-mãe e é a primeira coisa que começa a funcionar quando se liga o computador. A primeira etapa é a leitura das informações contidas no CMOS (Complementary Metal-Oxide-Semiconductor) com as configurações personalizadas pelo usuário no SETUP.

O BIOS é a primeira camada de software do sistema, um pequeno programa encarregado de reconhecer o hardware, realizar o boot, e prover informações básicas para o funcionamento do sistema. O BIOS é personalizado para cada modelo de placa mãe, não funcionando em nenhum outro.

3. Explique CMOS com suas palavras

CMOS é a abreviação de "Complementary Metal Oxide Semiconductor". O CMOS é uma pequena área de memória volátil, alimentada por uma bateria, que é usada para gravar as configurações do Setup da placa mãe.

Nos primeiros PC's todas as configurações acerca do hardware instalado, e demais configurações de inicialização do sistema, eram configurados através de jumpers, o que era um trabalho de certa forma complicado.

Para facilitar essas tarefas, foi criado o Setup, que permite configurar facilmente o sistema.

E o local onde as configurações desse Setup ficam armazenados é em uma pequena memória Ram de cerca de 128 bytes, chamada de CMOS.

4. Explique SETUP da BIOS com suas palavras

É um programa que nos permite configurar:

  • Várias opções acerca do hardware instalado;
  • Opções relacionadas ao desempenho do sistema;
  • senhas e etc;

As configurações do setup são essenciais para o funcionamento e bom desempenho do sistema, uma configuração rápida do setup pode tornar o sistema até 70% mais lento.

5. Explique a atualização da BIOS, como ela ocorre é porque é uma transação crítica

A BIOS é um programa que fica armazenado em chips de memória Flash Ram. O uso deste tipo de memória visa permitir que a BIOS seja modificada. A essa modificação damos o nome de upgrade ou atualização de BIOS.

Com o passar do tempo, novas técnologias vão surgindo tais como portas USB, barramente AGP e etc. A função do upgrade de BIOS é tornar o micro compatível com estes novos recursos.

É uma operação extremamente perigosa, pois ela consiste em:

  1. Baixar o programa de gravação dos dados e arquivo binário com o upgrade;
  2. Apagar os dados atuais da BIOS;
  3. Sobreescreve-los;

Este processo demora um pouco, cerca de alguns minutos e é isso que o torna tão crítico, afinal caso falte energia, se esbarre no botão de reset ou qualquer outro imprevisto que posso interromper o processo de atualização, a BIOS não funcionará mais e a placa mãe será inutilizada. Quando isso acontece, chamamos de brick ou dizemos que a BIOS está brickada.

8. Porque é possível detectar um malware mesmo sem conhece-lo?

É possível detectar um malware através da percepção de que o computador, não está funcionando como o esperado, apresentando sinais de infecção como pop-ups aparecendo aleatóriamente, programas "novos" desconhecidos, anuncios suspeitos ao navegar na internet, conexão mais lenta que o normal e etc.

9. Explique os principais tipos de malware

Vírus

Programas maliciosos que se multiplicam contaminando arquivos, registros de inicialização e tabelas de partição dos discos.

Para que o vírus seja ativado, é preciso abrir um arquivo contaminado ou inicializar o sistema com um disco contamindo.

Solução

Programas antivírus, detectam e eliminam vírus.

Cavalos de troia (Trojans)

São programas maliciosos que geralmente causam o mesmo tipo de problema que os vírus porém não se multiplicam, apenas tem alguma função nociva predeterminada.

Solução

Os programas anti-vírus também combatem os trojans porém para uma maior proteçam é indicado utilizar um programa específico, o antitrojan, que geralmente detecta e remove uma quantidade maior de cavalos de tróia que os antívirus

Worms

São como vírus mas se multiplicam automaticamente, geralmente exploram falhas de segurança em programas de gerenciamento de e-mails para se auto executarem e se espalharem rápidamente causando sérios prejuízos a nível mundial.

Solução

Programas antivírus e correções de segurança (patches) fornecidos pelos desenvolvedores dos sistemas operacionais e programas.

Spywares

São um tipo relativamente novo de malware utilizados por empresas de marketing para fazer propagandas abusivas e coletar dados sobre os usuários, seus hábitos de navegação e etc.

Os Spywares geralmente são obtidos através de determinados programas (em geral os que exibem anúncios comerciais) através de scripts e plugins em páginas de internet.

Solução

Programas anti-spyware que detectam e removem os spywares e alguns também impedem a instalação dos mesmos através de diversos métodos.

Backdoor

É um método não documentado de acesso a sistemas, de maneira remota, permitindo que como o nome sugere, sirva de "porta dos fundos" para que um malware adentre no sistema e ai possa causar estragos reais ao sistema, usualmente trojans.

Embora tenham esses usos ruins, também podem ser usados de forma legítima por fabricantes para restaurar acessos.

Solução

Deixar habilitado o firewall, além de mante-lo bem configurado. Usar um sistema de prevenção contra intrusos e um anti-malware pode ajudar também.

Ransomware

Ransomware é um tipo de código malicioso que torna inacessíveis os dados armazenados em um equipamento, geralmente usando criptografia, e que exige pagamento de resgate (ransom) para restabelecer o acesso ao usuário.

O pagamento do resgate geralmente é feito via bitcoins.

Eles podem se propagar das mais diversas formas, mas as mais comuns são:

  • através de e‐mails com o código malicioso em anexo ou que induzam o usuário a seguir um link;
  • explorando vulnerabilidades em sistemas que não tenham recebido as devidas atualizações de segurança.

Existem dois tipos de ransomware:

  • Ransomware Locker: impede que você acesse o equipamento infectado.
  • Ransomware Crypto: impede que você acesse aos dados armazenados no equipamento infectado, geralmente usando criptografia.

Além de infectar o equipamento o ransomware também costuma buscar outros dispositivos conectados, locais ou em rede, e criptografá‐los também

Solução

Para se proteger de ransomware você deve tomar os mesmos cuidados que toma para evitar os outros códigos maliciosos, como:

  • manter o sistema operacional e os programas instalados com todas as atualizações aplicadas;
  • ter um antivírus instalado;
  • ser cuidadoso ao clicar em links ou abrir arquivos.

Fazer backups regularmente também é essencial para proteger os seus dados pois, se seu equipamento for infectado, a única garantia de que você conseguirá acessá‐los novamente é possuir backups atualizados. O pagamento do resgate não garante que você conseguirá restabelecer o acesso aos dados.
